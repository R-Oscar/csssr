# Github issues viewer for csssr.ru

# Installation
Do:
```sh
$ git clone https://github.com/R-Oscar/csssr.git
$ npm i
```
Proceed to src and edit `config.js`
```sh
export const TOKEN = ''; // change
```

Fill in your personal Github API token

You're all set! Do:

```sh
$ npm start
```

& enjoy! :)
